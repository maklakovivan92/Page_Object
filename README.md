# Sprint 6 — UI автотесты для Яндекс Самокат (учебный тренажёр)

# Проект с автотестами UI для сайта Яндекс Самокат: оформление заказа, проверка логотипов и блок “Вопросы о важном”.
# Тесты написаны на **Python + Selenium + Pytest**, отчёты — **Allure**.

---

##  Что покрыто тестами

Всего: **12 тестов**

### 1 Логотипы (2 теста)
- Клик по логотипу **Самокат** → переход на главную страницу Самоката
- Клик по логотипу **Яндекс** → открытие **Дзен** в новой вкладке

### 2 Заказ самоката (2 теста)
- Успешный заказ через **верхнюю** кнопку “Заказать”
- Успешный заказ через **нижнюю** кнопку “Заказать”
(реализовано через параметризацию)

### 3 Вопросы о важном (FAQ) (8 тестов)
- Открытие каждого вопроса (0–7)
- Проверка, что текст ответа совпадает с ожидаемым

---

## Стек
- Python 
- Selenium
- Pytest
- Allure + allure-pytest

---

## Структура проекта

- `pages/` — Page Object (локаторы и методы страниц)
- `tests/` — тесты
- `data/` — тестовые данные (имена/адреса/телефоны/FAQ ответы)
- `conftest.py` — фикстуры (драйвер)

---

## Установка зависимостей

- pip install -r requirements.txt

---

## Запуск тестов

- pytest

---

## Запуск конкретного файла

- pytest tests/test_successful_order.py

---

## Прогон тестов с генерацией результатов

- pytest --alluredir=allure_result

---

## Открыть отчёт (локально)

- allure serve allure_results
